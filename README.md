---

## ✨ Overview

This is the official corporate website for Best Vision Agency. It is a fully responsive, bilingual (AZ / EN), dark-mode-ready single-page application designed to reflect the agency's premium positioning and creative standards.

Key highlights:
- 🌐 **Bilingual** — Azerbaijani and English with a seamless language toggle
- 🌙 **Dark / Light mode** — system-aware with persistent user preference
- 📬 **Live contact form** — multi-step form with EmailJS integration and real-time validation
- 🔍 **SEO-ready** — per-page meta tags, Open Graph, Twitter Card, sitemap, and robots.txt
- ⚡ **Buttery smooth** — page-wide Framer Motion animations throughout

## 🚀 Getting Started

**Prerequisites:** Node.js 18+

```bash
# 1. Clone the repository
git clone https://github.com/aykhanbadalov/best-vision-agency.git
cd best-vision-agency

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`.

```bash
# Build for production
npm run build

# Preview the production build
npm run preview
```

## 📁 Project Structure

```

├── components/       # Reusable UI components (Navbar, Footer, SEO, etc.)
```
